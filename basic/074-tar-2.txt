# Answer to exercise "7.4 Tar 2"

To list the contents you can use the command:
tar -tzf ~/package.tar.gz
Example output:
tmp/
tmp/dir1/
tmp/dir1/1.lst
tmp/dir1/2.txt
tmp/dir1/dir2/
tmp/dir1/dir2/1.txt
tmp/dir1/dir2/2.lst
