# Answer to exercise "1.3 VIM commands"
#
# Anything with '#' is a comment.
# The answer appears in line(s) of their own.

Beginning of line        = 0 (zero)
End of line              = $ (dollar)
Deleting a word          = dw
Deleting a line          = dd
Deleting a paragraph     = dap   (or: { then d})
Go to beginning of file  = gg
Go to end of file        = G
Copying line(s)          = yy (one line), 3yy (three lines), or: V then y
Copying a paragraph      = yap
Turn on/off line numbers = :set number   and   :set nonumber
