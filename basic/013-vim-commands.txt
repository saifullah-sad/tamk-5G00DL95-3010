Beginning of line        = 0
End of line              = $
Deleting a word          = dw
Deleting a line          = dd
Deleting a paragraph     = dap
Go to beginning of file  = gg
Go to end of file        = G
Copying line(s)          = yy / 3yy for 3 lines
Copying a paragraph      = yap
Turn on/off line numbers = :set number / :set nonumber
