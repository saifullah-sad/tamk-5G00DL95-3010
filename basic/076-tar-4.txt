# Answer to exercise "7.6 Tar 4"
#
tar -cjf ~/package.tar.bz2 -C ~ tmp
Explanation: "-c" is create archive, "-j" is compress using bzip2
"-f ~/package.tar.bz2" is the output file. and "-C ~ tmp" is to include tmp folder with relative paths.
